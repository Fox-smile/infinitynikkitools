# 简单的无限暖暖截图自动保存选择工具
该脚本由chatgpt完成，实现了在游戏中截图**自动保存**到文件夹时，进行监测并弹窗，由用户**自主选择**是否保存该截图
第一次使用时需要填写截图保存的路径，后续不需要

提供了py文件给小伙伴，希望可以进一步优化
